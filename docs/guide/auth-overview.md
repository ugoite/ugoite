# Authentication Overview Export

Use ugoite-core based output to inspect the currently implemented authentication contract.

## CLI

```bash
ugoite auth overview
```

The command prints a JSON snapshot generated by `ugoite-core` and includes:

- mandatory auth enforcement flags
- enabled provider capabilities (bearer / api_key)
- identity field contract
- supported client channels

## Source of Truth

- Runtime export: `ugoite_core.auth.export_authentication_overview()`
- Documentation contract: `docs/spec/security/authentication-overview.yaml`
- Consistency test: `ugoite-core/tests/test_auth_overview_consistency.py`
